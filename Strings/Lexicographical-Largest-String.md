# Lexicographically Largest String
Lexicographic - The way the words are arranged in a dictionary.  
A comes first for first position and second and every position will start with 'A'.  
In computer science, lexicographic order is a generalization of alphabetical order where strings are compared character by character using their Unicode (or ASCII) values.  

## Lexicographically Longest String
The one that would come last if you sorted them lexicographically.

What is we have 'abc' and 'abcd' then 'abcd' > 'abc'.
